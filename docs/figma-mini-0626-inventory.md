## 本地 `.fig` 可辅助确认

- `.fig` 是 Figma 私有二进制包，包含 `canvas.fig`、`thumbnail.png`、`meta.json` 和 `images/*`。
- `meta.json` 可确认导出时间与画布缩略图信息。
- `images/*` 可确认设计稿使用的图片素材类型：头像、山水路线、地图缩略、传统文化、餐食、音频/空状态插画、tabbar 截图等。
- `canvas.fig` 中无法直接搜索到可用节点名或完整布局参数，因此不能替代 MCP 参数。
