# 文字笔记定位 H5 设计规格

> **Historical / point-in-time:** 本文记录 2026-07-20 的 H5 视觉确认方案，不代表生产实现已经发布。
> **Current authority:** 以当前 `prototype/text-note-h5/` 原型代码和后续获批的生产设计为准。

## 目标

在不打断纯文字写作的前提下，复用图文发布已有的地点语义，让作者可以为文字笔记选填一个发布地点。

## 交互

- 第一步仍只有标题和正文，不出现地点入口。
- 第二步在封面样式下方增加“发布地点（选填）”工具条。
- 未选择时显示“设置地点”；点击后打开底部地点选择层。
- 选择地点后工具条显示地点名，点击主体可替换，点击删除按钮可清空。
- H5 使用固定示例地点模拟选择，不访问浏览器或微信定位，不产生云端写入。
- 发布后的双列瀑布流不展示地点，避免破坏卡片节奏；详情正文下方展示地点胶囊。

## 验收

- 可完成未选地点发布。
- 可选择、替换和删除地点。
- 刷新预览页后草稿地点仍在；发布后详情展示地点。
- 地点不进入文字封面，也不增加瀑布流卡片高度。
