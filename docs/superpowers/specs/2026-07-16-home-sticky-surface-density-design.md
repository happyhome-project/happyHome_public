# 首页吸顶 Surface 色彩与密度设计

> **Historical / point-in-time:** 本设计记录 2026-07-16 用户针对首页搜索与标签吸顶区域的视觉批注，仅用于交付追溯。
> **Current authority:** 以[文档权威索引](../../README.md)、当前首页代码和自动化测试为准。

## 目标

让社群标题、格言、搜索外围与主题标签外围共享参考图中的浅青绿色调，同时压缩搜索和标签两个背景框的垂直留白。现有两阶段原生 sticky 行为、搜索胶囊高度、标签文字字号与行高必须保持不变。

## 色彩

参考图取样显示标题区约为 `#d5f4f3`，格言中段约为 `#def4f4`，格言下沿约为 `#e6f4f6`。首页采用同一冷青色系：

- 顶部与标题以 `#d5f4f3` 为基底，并保留右上角柔和薄荷高光。
- 格言中段过渡到 `#def4f4`。
- 搜索与标签 sticky 外层统一使用 `#e6f4f6`，由一个页面级 CSS 变量管理，保证两块颜色严格一致。
- 搜索胶囊继续为白色，绿色搜索按钮、文字颜色与选中态不变。

## 密度与几何

- 搜索胶囊保持 `90rpx`；外层上下 padding 从 `24rpx` 降为 `4rpx`，总高度从 `138rpx` 降为 `98rpx`。
- hero 向下延伸量、负 margin 和 tabs sticky 偏移同步从 `138rpx` 改为 `98rpx`，因此搜索与 tabs 固定后仍紧贴且不重叠。
- 主题标签文字继续使用 `28rpx / 40rpx`；标签内层上 padding 从 `20rpx` 降为 `4rpx`，下 padding 保持 `18rpx`，为红色选中下划线保留空间。
- 不修改 DOM、滚动容器、`position: sticky`、z-index、点击逻辑或帖子流。

## 验收

- 静态契约锁定共同 surface 色、`98rpx` 搜索高度派生关系和标签文字行高。
- H5 smoke 验证搜索与 tabs 背景色一致、搜索总高约 51px、两阶段固定坐标仍紧贴、回滚仍释放。
- 小程序完整单测、H5 构建、微信构建、治理和文档检查通过。
