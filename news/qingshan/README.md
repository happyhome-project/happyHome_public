## 使用说明

每周目录按候选新闻主题归类，保留时间、发言人和聊天摘录，方便后续写成 `青山小新闻` 帖子。

每周目录还包含 `raw-excerpts.tsv`，按时间保存该周可读文本、引用、链接标题和视频号标题，便于后续回查原话；空图片、纯视频、表情等无文本内容不会出现在摘录表中。

优先级说明：

- A：很适合直接写成小新闻，有事件、有温度、有群内回应。
- B：可写，但最好补图、补视频号素材，或再确认细节。
- C：更像备忘、活动通知或生活信息，适合沉淀但不一定单独成新闻。

素材说明：

- `type=49, subType=51` 多为视频号/视频卡片，可作为后续视频组候选素材。
- `type=3` 为图片，`type=43` 为视频；本轮只做新闻线索整理，没有下载媒体文件。
- 群聊原话只是素材摘录，发布前建议做必要的隐私和语气处理。
